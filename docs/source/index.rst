Lambdify
====================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Lambdify is a simplified AWS Lambda Development for JVM languages like Kotlin and Java itself.
It will help you to design an easily maintable serverless application with low memory footprint.
